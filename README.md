# WorkTrack — Работни Часове PWA

Прогресивно уеб приложение (PWA) за проследяване на работни часове и заплата.

## 🚀 Бърз старт за GitHub Pages

1. Качи всички файлове в GitHub репозиторий
2. Отиди в **Settings → Pages**
3. Source: **Deploy from a branch → main → / (root)**
4. Отвори `https://<твоят-user>.github.io/<repo-name>/`

## ✨ Функции

- 📊 **Табло** — общо часове, заплата, работни дни, среден дневен брой
- 📅 **Календар** — кликни на ден за 6, 8, 12 или персонализирани часове
- 📈 **Графики** — стълбовидна и линейна диаграма (Chart.js)
- 📁 **История** — преглед по месеци с детайлна таблица
- ⬇️ **Експорт** — CSV и PDF (jsPDF)
- 🌙 **Тъмен/светъл режим** (запомня се)
- 📱 **Инсталируемо на телефон**, работи офлайн

## 💾 Поверителност

Всички данни остават в **localStorage** на браузъра ти. Нищо не се изпраща никъде.
